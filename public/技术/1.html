Rust 所有权机制

所有权是Rust的核心特性：

fn main() {
    let s = String::from("hello");
    takes_ownership(s);
    // println!("{}", s); // 这里会报错
}

所有权规则

  1. 每个值有且只有一个所有者
  2. 当所有者离开作用域，值被丢弃
  3. 借用分为可变和不可变引用
