# 现代CSS技术

## Grid布局示例
```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}
```

## 响应式断点
| 设备 | 宽度 | 列数 |
|------|------|------|
| 手机 | <768px | 1 |
| 平板 | 768-1024px | 2 |
| 桌面 | >1024px | 3 |
