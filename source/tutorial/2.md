# Git版本控制教程

## 基础命令
```bash
git add .
git commit -m "feat: 添加新功能"
git push origin main
```

## 分支管理策略
| 分支类型 | 用途 | 生命周期 |
|----------|------|----------|
| main | 生产环境 | 永久 |
| develop | 开发集成 | 长期 |
| feature | 功能开发 | 短期 |
| hotfix | 紧急修复 | 极短期 |
