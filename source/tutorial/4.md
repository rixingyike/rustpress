# 数据库优化技巧

## 索引优化
```sql
CREATE INDEX idx_user_email ON users(email);
CREATE INDEX idx_order_date ON orders(created_at);
```

## 查询优化策略
| 优化类型 | 描述 | 效果 |
|----------|------|------|
| 索引优化 | 创建合适索引 | 查询速度提升10-100倍 |
| 查询重写 | 优化SQL语句 | 减少不必要计算 |
| 分库分表 | 数据水平拆分 | 解决单表数据量过大 |
