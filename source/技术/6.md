# Rust 泛型编程

## 泛型函数示例
```rust
fn largest<T: PartialOrd>(list: &[T]) -> &T {
    let mut largest = &list[0];
    for item in list {
        if item > largest {
            largest = item;
        }
    }
    largest
}
```

## 特性约束
- 使用 trait bounds
- where 子句简化
- 关联类型应用

| 泛型类型    | 常用 trait       |
|------------|-----------------|
| 数值计算    | Add/Mul         |
| 比较操作    | PartialOrd      |
| 格式化输出  | Display/Debug   |
