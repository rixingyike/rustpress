# Rust 宏系统

## 声明宏示例
```rust
macro_rules! vec {
    ($($x:expr),*) => {
        {
            let mut temp_vec = Vec::new();
            $(temp_vec.push($x);)*
            temp_vec
        }
    };
}
```

## 过程宏应用
- 派生宏实现序列化
- 属性宏创建路由处理器
- 函数宏实现DSL语法
