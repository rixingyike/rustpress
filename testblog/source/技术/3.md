# Rust 错误处理

## Result 与 Option
```rust
fn read_file(path: &str) -> Result<String, std::io::Error> {
    std::fs::read_to_string(path)
}

fn find_user(id: u32) -> Option<String> {
    Some("John".to_string())
}
```
