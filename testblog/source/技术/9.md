# Rust 属性编程

## 过程宏应用
```rust
#[derive(Serialize)]
struct User {
    id: u32,
    name: String,
}
```

## 常用属性
| 属性            | 用途                 |
|-----------------|----------------------|
| #[derive(Debug)] | 自动实现Debug trait |
| #[test]         | 单元测试标记          |
| #[allow(unused)] | 抑制未使用警告        |
