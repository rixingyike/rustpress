# Rust 生命周期

## 函数签名标注
```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}
```

## 结构体应用
- 字段引用需要显式生命周期
- 生命周期省略规则
- 静态生命周期`'static`

| 场景          | 是否需要标注 |
|---------------|-------------|
| 单一输入引用  | ❌           |
| 多个输入引用  | ✅           |
| 返回引用      | ✅           |
